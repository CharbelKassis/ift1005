Installation de WinCDEmu

  1. Pour l'installation sous Windows, aller sur le site suivant http://wincdemu.sysprogs.org/download/ puis cliquer sur le boutton download.
  2. Une fois téléchargé, double cliquez sur l'exécutable.
  3. Cliquez sur install.

Utiliser le logiciel

  1. Double cliquer sur un fichier .iso ou équivalent. Une fenêtre s'ouvrira.
  2. Choissisez le nom du lecteur virtuel, puis le type de disque (facultatif).
  3. Finalement selectionner OK.

Vous pouvez désormais parcourir le contenu du disque virtuel en allant dans My Computer. (Ordinateur)

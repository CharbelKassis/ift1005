Qu'est-ce que WinCDEmu?

WinCDEmu est un programme qui est principalement utilisé pour émuler un lecteur de disques (CD / DVD / Bluray). Il est particulierement utile si vous possédez des fichiers .iso ou equivalents, c'est-à-dire des fichiers qui sont prêts à être gravés sur un disque et que:
  • Vous n'avez pas le lecteur/graveur nécessaire
  • Vous n'avez pas de disques
  • Vous ne voulez pas avoir plusieurs disques physiques pour des raisons d'espaces

Ce genre de programme est souvent utilisé en combinaisant avec d'autres programmes. Puisqu'il sert à lire des disques, il devrait existait une façon d'écrire des fichier .iso. On peut se servir de WinCDEmu pour extraire ces fichiers, effectuer des changements localement puis recréer un nouveau fichier .iso avec un autre programme.

Un exemple où ceci est utile: Ajouter des mises à jour à un disque d'installation Windows.
Lorsque vous téléchargez une image de Windows depuis le site de Microsoft, ce fichier ne contient aucuns pilotes hormis ceux nécessaires pour faire fonctionner l'ordinateur, donc il est impossible d'utiliser l'internet ou certains appareils qui utilisent des ports USB directement. Il faudrait d'abord que vous installer les pilotes de votre carte réseau pour accéder à l'internet. Ensuite, si vous ne voulez pas attendre une éternité que Windows Update trouvent les updates importantes, il suffit de les ajouter vous-mêmes pour qu'ils s'installent après Windows automatiquement. Avec WinCDEmu, on peut extraire tous les fichiers Windows, puis leur ajouter les mises à jour grâce à un programme puis recréer un nouveau .iso avec un autre logiciel.
